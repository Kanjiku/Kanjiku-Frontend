# Introduction

This is the frontend for a scanlation site.\
using VueJS 3 composition-api

# Installation

## Requirements
- node
- backend

# Running it

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

# Features

- Registration / Login / Logout
- Switching themes
